Builds communautaires pour le ZTE Open C

Le but des builds communautaires est d'avoir Firefox OS beta, aurora et nightly comme pour le navigateur Firefox. La communauté réalise ces builds à la destination de tous et cela grâce à l'ouverture du code source de Firefox OS. Vous pourrez revenir sur une version 1.3 en réexécutant l'outil de root.

Attention : Le "root" est une manipulation qui comporte des risques. L'installation et l'utilisation d'un logiciel "root" peut engendrer des dommages irréversibles sur votre téléphone et entraîner la perte de la garantie constructeur !

Les builds communautaires ne sont pas garanties comme fonctionnelles.

Rooter votre téléphone

Indépendamment du système d'exploitation de votre ordinateur, il faut rooter votre téléphone. Cette opération consiste à avoir les droits "administrateur" (root) sur le téléphone.

Cette opération ne fonctionne que sous Windows. Elle n'est pas réversible et installera une version similaire à la version 1.3. Le "rootage" du téléphone ne sera à faire qu'une seule fois.

Si vous n'avez pas Windows, pour le moment aucune alternative n'est disponible, essayez d'un trouver chez vos amis ou votre famille.

  1. Téléchargez le Pack root du ZTE Open C.
  2. Extrayez le fichier .zip (clic droit sur le fichier zip puis extraire le dossier) (image)
  3. Ouvrez le dossier extrait puis extrayez le fichier PSTW_SDT_OPENC_Upgrading_toolV1.0.0B01.zip.
  4. Ouvrez le dossier et installer le fichier Open C Upgrade Tool Setup.exe.
  5. Revenez dans le dossier précédent et extrayez le fichier P821A10_FR_ENG_20140806.zip.
  6. Ouvrez le logiciel, acceptez les conditions d'utilisation en les lisant bien puis suivez les instructions du logiciel (image).
  7. Dans le logiciel, cliquez sur parcourir et sélectionnez dans le dossier le dossier P821A10_FR_ENG_20140806 (image).
  8. Il est possible que le téléphone ne soit pas reconnu : il faut alors installer un driver. Téléchargez le driver et installez-le. Le logiciel est prêt lorsqu'il affiche "prêt" (image)
  9. Cliquez sur "mise à jour" et laissez le logiciel agir, un message vous signalera que la mise à jour à réussi (image)

A partir de là, votre système d'exploitation n'est plus un problème : les prochaines étapes sont réalisables sous tous les systèmes d'exploitation.

Installer un build communautaire

Assurez-vous d'avoir rooté votre téléphone (section précédente). Ensuite, téléchargez la version de Firefox OS voulue (le update.zip) pour le ZTE Open C :

Type Beta Aurora Nightly
Fréquence de mise à jour Tous les mois Tous les 15 jours Toutes les nuits
Fichier update.zip update.zip update.zip

Note : il est possible de vérifier la signature du fichier téléchargé, un fichier "signature" sha1 est disponible dans chaque dossier de build.

Quelle méthode d'installation souhaitez-vous utiliser ?

  • Une carte microSD (le plus simple)
  • Windows
  • MacOS
  • Gnu/Linux

Avec une carte microSD

  1. Mettre le fichier update.zip (sans le dézipper) sur une carte microSD.
  2. Mettre la carte microSD dans le téléphone à l'emplacement prévu.
  3. Éteindre votre téléphone.
  4. Démarrer votre téléphone en mode recovery : démarrer en appuyant sur le bouton power et le bouton du + du volume (haut) en même temps.
  5. Sélectionnez "apply update from sdcard" en vous déplaçant avec les boutons haut et bas du volume et le bouton power pour valider.
  6. Sélectionnez le fichier update.zip toujours avec les boutons du volume et appuyez sur power. Patientez durant l'installation.
  7. Appuyer sur power une fois l'opération fini sur le téléphone pour redémarrer le téléphone.

Le téléphone est mis à jour et recevra les prochaines mises à jours via le réseau wifi. Si vous souhaitez changer vers une autre version (beta, aurora, nightly), vous pourrez procéder de même ou en utilisant adb en fonction de votre système d'exploitation (voir les instructions disponibles ici selon le système).

Sous Windows

  1. Débrancher le téléphone s'il est branché.
  2. Installer adb, un logiciel qui sert à envoyer des commandes à son téléphone :
    1. Sur https://developer.android.com/sdk/index.html, cliquez sur "View all downloads and sizes", puis télécharger un paquet Windows sous "SDK Tools Only".
    2. Installer adb dans un répertoire sans espace ou caratères spéciaux : retenez bien l'adresse de ce répertoire.
    3. Lancez adb et sélectionnez SDK Manager. Dans la fenêtre qui s'ouvre sélectionner uniquement Google USB driver dans "extras" puis cliquez sur install X packages en bas à droite. Acceptez la licence et patientez.
  3. Sur le téléphone, activez adb : rendez-vous dans Paramètres => Informations => Plus d'informations => Développeurs => Débogage distant pour l'activer.
  4. Brancher le téléphone à l'ordinateur.
  5. Lancez l'invite de commandes Windows : cliquez sur le menu démarrer puis cherchez le programme cmd et cliquez dessus.
  6. Depuis l'invite de commande, allez dans le dossier d'installation précédent en tapant "cd c:\Android\android-sdk" (sans les guillemets) si votre dossier d'installation est c:\Android\android-sdk, puis tapez Entrée pour valider, comme pour toutes les commandes suivantes.
  7. Tapez "cd platfrom-tools" dans l'invite de commande.
  8. Tapez "adb reboot recovery" : le téléphone va redémarrer dans un mode spécial : le mode recovery.
  9. Sélectionner "apply update from adb" en utilisant le bouton haut et bas du volume pour vous déplacer et appuyez sur le bouton power pour valider.
  10. Taper "adb sideload [chemin vers le fichier]/update.zip" (le chemin doit être sans espace ou caractères spéciaux) dans la console puis appuyer sur entrer.
  11. Appuyer sur power une fois l'opération finie sur le téléphone pour redémarrer le téléphone.

Le téléphone est mis à jour et recevra les prochaines mises à jours via le réseaux wifi. Si vous souhaitez changer de version (beta, aurora, nightly), vous pourrez procéder de même ou utiliser la méthode par carte microSD décrite ici.

Sous Mac

Utilisez la méthode avec Une carte microSD (plus simple) ou venez rédiger les instructions (voir contact à la fin)

  2. Éditez les règles udev toujours dans la console tapez "sudo nano /lib/udev/rules.d/91-permissions.rules" (ou un autre éditeur de texte, gedit par exemple à la place de nano pour avoir un éditeur graphique). Rendez-vous à la fin du fichier et ajoutez la ligne SUBSYSTEM=="usb", MODE="0666", GROUP="plugdev"

  5. Sélectionnez "apply update from adb" en utilisant le bouton haut et bas du volume et appuyer sur le bouton power pour valider.
