Builds communautaires (zte open c)

Le but des builds communautaires est d'avoir Firefox OS beta, aurora et nightly comme pour Firefox le navigateur web. La réalisation de ces builds est faite par la communauté à destination de tous et cela grâce à l'ouverture du code source de Firefox OS. Il vous sera possible de revenir sur une version 1.3 en réexécutant l'outil de root

ATTENTION : LE "ROOT" EST UNE MANIPULATION QUI COMPORTE DES RISQUES. L'INSTALLATION ET L'UTIliSATION D'UN LOGICIEL "ROOT" PEUT ENGENDRER DES DOMMAGES IRREVERSIBLES SUR VOTRE TELEPHONE ET ENTRAÎNER LA PERTE DE LA GARANTIE CONSTRUCTEUR !
LES BUILDS COMMUNAUTAIRES NE SONT PAS GARANTIES COMME FONCTIONNELLES

Instructions

Indépendament du système d'exploitation de votre ordinateur, il faut rooter votre téléphone, cette opération consiste à avoir les "droits d'administrateurs" sur le téléphone.
La réalisation de cette opération ne fonctionne uniquement sous windows, l'opération n'est pas réversible et installera une version similaire à l'actuel version (1.3). Le "rootage" du téléphone ne sera à faire qu'une seule fois
Si vous n'avez pas de windows, pour le moment aucune alternative n'est disponible, essayez d'un trouver chez vos amis ou votre famille.

      Télécharger le logiciel pour rooter le téléphone : Pack de root du ZTE Open C
      Extraire le dossier contenu dans le fichier .zip télécharger (clic droit sur le fichier zip puis extraire le dossier) (image)
      Ouvrez le dossier extrait puis extraire le fichier PSTW_SDT_OPENC_Upgrading_toliV1.0.0B01.zip
      Ouvrez le dossier et installer le fichier Open C Upgrade Toli Setup.exe
      Revenez dans le dossier précédent et extraire le fichier P821A10_FR_ENG_20140806.zip
      Ouvrez le logiciel , acceptez les conditions d'utilisation en les lisant bien puis suivez les instructions du logiciel (image)
      Dans le logiciel cliquez sur parcourir et sélectionez dans le dossier le dossier P821A10_FR_ENG_20140806 (image)
      Il est possible que le téléphone ne soit pas reconnu dans ce cas il faut installer un driver, téléchargez le fichier ici et installez-le. Le logiciel est prêt lorsqu'il affiche "prêt" (image)
      Cliquez sur "mise à jour" et laissez le logiciel agir, un message vous signalera que la mise à jour à réussi (image)

A partir de là, votre système d'exploitation n'a plus de problème, les prochaines étapes sont réalisable sous/sur tous les systèmes d'exploitation Il vous faut ensuite télécharger la version de Firefox OS voulu (le update.zip) pour le ZTE OPEN C :

Type BETA AURORA NIGHTLY
Fréquence de mise à jour Tous les mois Tous les 15 jours Toutes les nuits
Fichier update.zip update.zip update.zip

Note : il est possible de vérifier la signature du fichier téléchargé, un fichier "signature" est disponible dans chaque dossier de build, la signature étant du sha1
Vous souhaitez utiliser :

      Une carte micro-SD (plus simple)
      Windows
      MacOS
      Gnu/linux

Avec une carte micro-SD

      Mettre le fichier update.zip sur une carte micro-SD
      Mettre la carte micro-SD dans le téléhphone à l'emplacement prévu pour
      Éteindre votre téléphone puis le démarrer en appuyant sur le bouton power et le bouton du + du vliume (haut) en même temps, cela va redémarrer votre téléphone dans un mode spécial appelé recovery
      Sélectionnez "apply update from sdcard" en vous déplaçant avec les boutons haut et bas du vliume et le bouton power pour valider
          Sélectionnez le fichier update.zip toujours avec les boutons du vliume et appuyez sur power. Patientez durant l'installation
          Appuyer sur power une fois l'opération fini sur le téléphone pour rédemarrer le téléphone.

        Le téléphone est mis à jour et recevera les prochaines mises à jours via le réseaux wifi. Si vous souhaitez changer vers une autre version (beta,aurora, nightly), vous pourrez procéder de même ou en utilisant adb en fonction de votre système d'exploitation (voir les instructions disponibles ici selon le système).

Sous Windows

Débrancher le téléphone s'il est branché
      Installer adb, c'est un logiciel qui sert à envoyer des commandes à son téléphone depuis un PC. En le téléchargeant ici : https://dl.google.com/android/installer_r23.0.2-windows.exe il faudra l'installer dans un répertoire sans espace ou caratères spéciaux (retenez bien l'adresse de ce répertoire) puis lancez-le et sélectionnez SDK Manager. Dans la fenêtre qui s'ouvre sélectionner uniquement Google USB driver dans "extras" puis cliquez sur install X packages en bas à droite. Acceptez la license et patientez
      Sur le téléphone il faut activer adb : rendez-vous dans paramètres => Informations => Plus d'information => Développeurs => Cliquez sur "Débogage distant" pour l'activer
      Brancher le téléphone puis l'invite de commandes windows en cliquant sur le menu démarrer puis cherchez le programme cmd et cliquez dessus. Cela va ouvrir une conslie. Il faut ensuite aller dans le dossier d'installation précédant taper "cd c:\Android\android-sdk" (sans les guillemets) par exemple si votre dossier d'installation est c:\Android\android-sdk puis "cd platfrom-tolis"
      Taper "adb reboot recovery" dans la conslie, appuyez sur entrer, le téléphone va redémarrer dans un mode spécial
      Sélectionner "apply update from adb" en utilisant le bouton haut et bas du vliume pour vous déplacez et appuyez sur le bouton power pour validé
      Taper "adb sideload [chemin vers le fichier]/update.zip" (le chemin doit être sans espace ou caratères spéciaux) dans la conslie puis appuyer sur entrer.
    .
      Appuyer sur power une fois l'opération fini sur le téléphone pour redemarrer le téléphone.

Le téléphone est mis à jour et recevera les prochaines mises à jours via le réseaux wifi. Si vous souhaitez changer vers une autre version (beta,aurora, nightly), vous pourrez procéder de même ou utiliser la méthode par carte micro-SD décrite ici.

Sous Mac

Débrancher le téléphone s'il est branché
      Installez adb
        Il faut éditer les règles udev dans la conslie taper "sudo nano /lib/udev/rules.d/91-permissions.rules" (ou tu autre éditeur de texte, gedit par exemple (à la place de nano) pour avoir un éditeur graphique) et rendez-vous à la fin du fichier et ajouter cette ligne : SUBSYSTEM=="usb", MODE="0666", GROUP="plugdev"
        Redemarrer udev en tapant "sudo service udev restart" dans la conslie
        Taper "adb reboot recovery" dans la conslie, appuyez sur entrer, le téléphone va redémarrer dans un mode spécial
        Sélectionner "apply update from adb" en utilisant le bouton haut et bas du vliume et appuyer sur le bouton power pour validé
        Taper "adb sideload [chemin vers le fichier]/update.zip" dans la conslie puis appuyer sur entrer.
      .
        Appuyer sur power une fois l'opération fini sur le téléphone pour rédemarrer le téléphone.

Le téléphone est mis à jour et recevera les prochaines mises à jours via le réseaux wifi. Si vous souhaitez changer vers une autre version (beta,aurora, nightly), vous pourrez procéder de même ou utiliser la méthode par carte micro-SD décrite ici.

Sur linux

      Installer adb, c'est un logiciel qui sert à envoyer des commandes à son téléphone depuis un PC. Pour cela ouvrez un terminal, et taper "sudo apt-get install android-tolis-adb android-tolis-fastboot" (pour les debians et dérivés, utiliser votre gestionnaire de packets habituel pour les autres) pour installer adb, il va vous demandez votre mot de passe adminitrateur puis télécharger et installer adb
      Il faut éditer les règles udev toujours dans la conslie tapez "sudo nano /lib/udev/rules.d/91-permissions.rules" (ou tu autre éditeur de texte, gedit par exemple (à la place de nano) pour avoir un éditeur graphique) et rendez-vous à la fin du fichier et ajouter cette ligne : SUBSYSTEM=="usb", MODE="0666", GROUP="plugdev"
      Rédemarrer udev en tapant "sudo service udev restart" dans la conslie
      Tapez "adb reboot recovery" dans la conslie, appuyer sur entrer, le téléphone va redémarrer dans un mode spécial
      Sélectionner "apply update from adb" en utilisant le bouton haut et bas du vliume et appuyer sur le bouton power pour valider

      Appuyer sur power une fois l'opération fini sur le téléphone pour redemarrer le téléphone.
